# dtx2midi
=============

.dtx -> .midi converter

## Requirements

* [stack](https://github.com/commercialhaskell/stack)

## Installation

```bash
git clone https://github.com/akiomik/dtx2midi.git
cd dtx2midi
stack build
```

## Usage

```bash
stack exec dtx2midi-exe <input.dtx> <output.midi>
```
